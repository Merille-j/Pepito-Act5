# Pepito — Activity 5: AJAX Portfolio

A personal portfolio website in compliance with academic curriculum of Web Development.

---

## Folder Structure

```Pepito-Act5/
├── index.html           # Home — hero, quick-nav cards
├── about.html           # About — portrait, bio, experience
├── work.html            # Work — projects fetched via fetch() + live search
├── skills.html          # Skills — animated bars, tools, services
├── contact.html         # Contact — AJAX form submit, no page reload
├── data/
│   └── projects.json    # JSON data source for all projects
├── images/              # Real project screenshots + profile photo
├── css/
│   └── style.css        # Shared stylesheet + spinner/toast styles
└── js/
    ├── main.js          # Nav, scroll reveal, skill bars, mobile drawer
    └── ajax.js          # All async logic — fetch, search, filter, toast
```
